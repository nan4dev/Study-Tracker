# Study Tracker

A command-line Python application for tracking study sessions and viewing study statistics.

## Features

- Add study sessions
- Save sessions permanently using text files
- Categorize study topics
- View study history
- Calculate total study time
- Track average session length
- Find longest study session
- View category-wise study totals
- Interactive menu system

## Technologies Used

- Python
- Git
- GitHub
- VS Code
